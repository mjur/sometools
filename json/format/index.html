FAQ

Can I sort JSON keys?

Yes, enable the 'Sort keys' option to alphabetically sort all object keys.

What indentation options are available?

You can choose 2 spaces, 4 spaces, or tabs for indentation.

Can I minify JSON?

Yes, use the 'Minify' button to remove all whitespace and create compact JSON.
