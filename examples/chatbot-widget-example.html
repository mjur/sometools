Chatbot Widget Example

How to Include the Chatbot Widget

This page demonstrates how to include the chatbot widget on any page.

Method 1: ES Module Import (Recommended)

<script type="module">
  // Load WebLLM bundle first (required)
  try {
    await import('/js/tools/bundled/webllm-bundle.js');
  } catch (e) {
    console.log('WebLLM bundle not found');
  }
  
  // Initialize the widget
  import { initChatbotWidget } from '/js/utils/chatbot-widget.js';
  initChatbotWidget();
</script>

Method 2: Using Script Tags

<!-- Load WebLLM bundle first -->
<script type="module" src="/js/tools/bundled/webllm-bundle.js"></script>

<!-- Load and initialize widget -->
<script type="module">
  import { initChatbotWidget } from '/js/utils/chatbot-widget.js';
  initChatbotWidget();
</script>

Requirements

  • The WebLLM bundle must be loaded before initializing the widget
  • The widget requires WebGPU support in the browser
  • All dependencies (CSS, UI utilities) are automatically included
