Transformers.js requires ONNX models, but most AI detection models on Hugging Face don't have ONNX versions.

The tool will attempt to load models, but most will fail because they lack ONNX exports.

Solution: To use this tool effectively, you need to:

  1. Convert an AI detection model to ONNX format using Python and Optimum: 
    python -m scripts.convert --quantize --model_id roberta-base-openai-detector
  2. Host the converted model files locally (or on a server)
  3. Update the model configuration in the code to point to your local model

Alternatively, the Hugging Face space version likely uses server-side processing or pre-converted models.
