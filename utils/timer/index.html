FAQ

How does the stopwatch work?

The stopwatch counts up from zero, tracking elapsed time. Click Start to begin, Pause to pause, and Reset to return to zero.

How does the timer work?

Set the hours, minutes, and seconds, then click Start. The timer counts down to zero and will alert you when it reaches zero.

Does it work in the background?

The timer/stopwatch continues running even if you switch tabs, but may pause if the browser tab is inactive for extended periods.
