FAQ

What is IP lookup?

IP lookup provides information about an IP address, including whether it's IPv4 or IPv6, if it's private/public, and basic network information.

What is a subnet calculator?

A subnet calculator helps you calculate network addresses, broadcast addresses, host ranges, and subnet masks for IP networks.

Is my IP address sent to a server?

No. All calculations happen locally in your browser. However, to get your public IP address, a request may be made to a third-party service.
